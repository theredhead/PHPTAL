Introduction

top

PHPTAL is a PHP implementation of ZPT work. To be short, PHPTAL is a XML/XHTML template library for PHP.

While most web developpers continue to use ASP/JSP/PHP tags as the core language of their templates, the Zope community came with a refreshing idea named TAL. The idea was to move presentation actions inside XHTML attributes instead of using plain tags or elements.

Some PHPTAL advantages and features :

  • enforce the separation between logic and presentation (the holy grail),
  • warn you if you forgot to close an html tag or marlformed your template,
  • no more htmlentities,
  • quite clean and readable templates,
  • ability to insert sample text inside template to preview template result without PHP backend,
  • integrates quite well with WYSIWYG HTML editors,
  • data abstraction using xpath like system,
  • cool html macro system,
  • integrated internationalization system,
  • ability to replace xpath system with php expressions,
  • nearly no speed loss (PHP5 with php expression mode active).
  • template source filter interface
  • more than four years of ZPT community support (see zope websize)

Some PHPTAL drawbacks :

  • a dependency for your application,
  • some tal, metal, i18n and phptal attributes to understand,
  • you'll have to produce clean XHTML.

From now on prefer using the PHP5 version of PHPTAL (works only with PHP5).

PHP5 is stable and usually smarter than PHP4. We use it in my company on complex production websites (intensively used) without any problem.

A free PHP5 accelerator is also working well with PHP5.

Please forget PHP4.

Installation procedure

You'd better have the PEAR utility installed to use these packages, if you don't know PEAR (as too many PHP programmers), you should take the time to learn about it and how to use it. It really worth the time you will spend on it.

The usual way of installing packages with pear is : 

pear install {downloadedfile}

PHPTAL for PHP5

  • PHPTAL-${VERSION}.tar.gz 
pear install http://phptal.motion-twin.com/files/PHPTAL-${VERSION}.tar.gz

PHPTAL for PHP4

You will need to download and install packages contained in this directory.

Subversion repository

You can access latest PHPTAL source code using subversion. Using this code in production is risky. 

svn co https://svn.motion-twin.com/phptal/trunk phptal

If you are willing to send me patches, please use svn diff to easier the communication.
